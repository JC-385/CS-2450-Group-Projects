This project is a simple virtual machine that inteperets basicML.
To run the program open a command console and type out python Main,
or run inside a code editor with a console like Visual studio or visual studio code.
the program input is formated to open a file with 4 digits per line
the first 2 are the operator and the last 2 are the operand
the only requirement is to have the most recent python downloaded.
To run the test, you have to run this command in the terminal: pytest UVSim/Tests.py
